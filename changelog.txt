YouTube Comment Filter Changelog

Version 2.3 (14 January 2018)
Added ability to block users.

Version 2.2.1 (2 October 2017)
Faster loading.

Version 2.2.0 (1 September 2017)
Flairing comments by default now.
Fade in on mouseover on a flaired comment.
Filtering rechecks comments (since deleted comments might come back after filtering).
Added some extra spam checks.

Version 2.0.1 (31 August 2017)
Fixed SPAM flair.

Version 2.0 (30 August 2017)
Added support for the new YouTube layout (old version not supported anymore, use 1.0.3 instead).
Increased performance and speed.
Fixed counters from breaking.

Version 1.0.3 (20 April 2017)
Fixed total number of comments showing a . or , when less than 1000 comments.
Fixed Spam flag name.

Version 1.0.2 (20 April 2017)
Fixed a bug in which the filtering stopped after removing one comment.

Version 1.0.1 (20 April 2017)
Improved the spam counter to a simpler implementation.
Improved debug logging.
Fixed second comment removal causing the comments counter not being updated.

Version 1.0.0 (19 April 2017)
Fixed: All comments were being removed.
Added options for the user to enable/disable.
Added debug option (you're able to view which comments were removed for a specific rule in the browser's Console if turning this on.
Some code improvements, pretty minor.

Version 0.0.4 (20 December 2016)
Added a few filters.
Added general minimal word count.
Removed filter option flags.

Version 0.0.3 (14 September 2016)
Comment counter now changes if comments are found and deleted.
Added spam counter shown next to the comment counter.

Version 0.0.2 (25 August 2016)
Fixed YouTube changing code, which broke this script.
Added working comment counter (but copies the SPAM flag after loading more comments (nothing breaking)).

Version 0.0.1 (16 July 2016)
First working draft (internal release).
